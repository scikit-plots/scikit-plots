What is a transformer model?
A transformer is a neural network that uses self-attention to process sequences in parallel — rather than step-by-step. It powers GPT, Claude, Gemini, and most modern LLMs.

How does self-attention work?
Each token computes query, key, and value vectors. Attention scores are dot-products of queries vs all keys, softmaxed to weights, then summed over values — letting every position attend to every other simultaneously.
