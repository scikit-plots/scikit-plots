    Downloads & Installation
    ========================

    Install the latest version of Scikit-Plots via:

    - `PyPI: scikit-plots <https://pypi.org/project/scikit-plots>`_
    - `Conda-Forge: scikit-plots <https://anaconda.org/conda-forge/scikit-plots>`_
    - `Nightly-Builds: scikit-plots-wheels-staging-nightly <https://anaconda.org/scikit-plots-wheels-staging-nightly/scikit-plots>`_
